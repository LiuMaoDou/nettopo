# NetTopo

A full-stack network topology visualization tool built with React + G6 v5 and FastAPI.

## Features

- Interactive topology canvas вЂ” drag, zoom, search, highlight
- Per-link interface labels with directional utilization: `Gi0/1(71%)` в†ђв†’ `ge-0/0/0(60%)`
- Device status coloring (up / warning / down)
- Edge color reflects link utilization (green в†’ yellow в†’ orange в†’ red)
- Multiple layout algorithms: Force, Dagre, Circular, Radial, Combo
- Import topology from JSON or CSV (devices + links as separate files)
- Export canvas as PNG
- Real-time updates via WebSocket
- Demo data at three scales: 20 / 200 / 1500 devices

## Quick Start

**Mac**

```bat

```

**Manual**
```bash

```

## Import Format

### JSON

Node `id` is optional вЂ” auto-generated by slugifying the label (`Core-Router-1` в†’ `core-router-1`). Edge `source`/`target` must match the node label (or its slugified form).

### CSV
Two files required (download from **еҮәе…ӨжЁҰжқү в†’ CSV**):

**devices.csv**
```
label,type,ip,mac,vendor,model,location,group,status
Core-Router-1,router,10.0.0.1,,Cisco,ASR9000,DC-A / Rack-1,g-core,up
```

**links.csv**
```

Core-Router-1,Dist-SW-1,Gi0/1,ge-0/0/0,10000,0.71,0.60,fiber,up
```

Supported device types: `router` `switch` `firewall` `server` `ap` `endpoint`
